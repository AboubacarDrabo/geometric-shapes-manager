# Geometric Shapes Manager 🔷

A Java OOP project built as part of my college coursework (Object-Oriented Programming 2 — Collège Ahuntsic).

## Description

This application allows users to create, display and manage geometric shapes through a graphical interface (Java Swing). It demonstrates core OOP concepts such as inheritance, polymorphism, and encapsulation.

## Features

- Create and display multiple shape types: **Circle**, **Square**, **Rectangle**, **Hexagon**
- Interactive **Swing GUI** with buttons and visual feedback
- Input validation with exception handling
- Polymorphic design — all shapes extend a common `Polygone` base class

## Class Structure
Polygone (abstract base class)
├── Cercle
├── Carre
├── Rectangle
└── Hexagone
## Concepts Used

- Object-Oriented Programming (inheritance, encapsulation, polymorphism)
- Java Swing (GUI, ActionListeners, JComboBox)
- Exception handling (`try/catch`)
- ArrayList for managing multiple objects
- UML class diagram design
